# GetaSong

## Overview

A simple music API that uses JWT auth

## Installation

```
$ git clone https://github.com/SowmyaAji/GetaSong.git
$ cd to the cloned repository
$ pipenv shell
$ pip install -r requirements.txt
$ python manage.py migrate

```

## Testing

```
$ python manage.py test
```
